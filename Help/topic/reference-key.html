キーボードショートカット

Special Launch は、ほとんどの動作がキーボードで操作可能です。キーボードショートカットをおぼえることで起動したいプログラムをすばやく選択したり、ボタンの編集を簡単に行うことが可能です。

通常 Enter
カーソル位置のボタンを押します。
Space
メニューを開きます。
カーソル移動 ↑↓←→
カーソルを上下左右に移動します。
PageUp／PageDown
カーソルを 1 画面分上下に移動します。
Home／End
カーソルを現在の行の最初／最後に移動します。
Ctrl + Home／End
カーソルを最初／最後に移動します。
ボタン名
そのボタンにカーソルを移動します。
ボタン編集 Ctrl + E
［ボタンの編集］ダイアログボックスを開き、現在のパッドのボタンを編集します。
Ctrl + A
［ボタンの追加］ダイアログボックスを開き、カーソル位置にボタンを追加します。
Alt + Enter
［ボタンの変更］ダイアログボックスを開き、カーソル位置のボタンを変更します。
Alt + ボタンをクリック
〃
Shift + Enter
カーソル位置のボタンに登録されているリンク先の 1 つ上のフォルダを開きます。
Shift + ボタンをクリック
〃
Ctrl + X
カーソル位置のボタンをクリップボードに切り取ります。
Ctrl + C
カーソル位置のボタンをクリップボードにコピーします。
Ctrl + V
クリップボードのボタンをカーソル位置に貼り付けます。
Ctrl + Space
カーソル位置に空白を挿入します。
Ctrl + Enter
カーソル位置で改行します。
Delete
カーソル位置のボタンを削除します。
Backspace
カーソル位置の 1 つ前のボタンを削除します。
Shift + Delete
カーソル位置の 1 つ後のボタンを削除します。
ボタングループ Ctrl + 1〜9
ボタングループを切り替えます。
Ctrl + PageUp／PageDown
前後のボタングループに切り替えます。
パッド Ctrl + P
現在のパッドの［パッドの設定］ダイアログボックスを開きます。
Ctrl + Tab
次のパッドをアクティブにします。
Ctrl + Shift + Tab
前のパッドをアクティブにします。
Esc
現在のパッドを画面の四辺に隠します。
その他 Ctrl + R
［指定して実行］ダイアログボックスを開きます。
Ctrl + O
［全体の設定］ダイアログボックスを開きます。
Ctrl + Q
Special Launch を終了します。
